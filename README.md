# Simple Renamer

Simple Renamer is an open-source file renaming utility developed using the .NET framework, specifically utilizing WPF (Windows Presentation Foundation). Inspired by the functionality of pyRenamer, this application provides users with an intuitive interface to easily rename multiple files and folders in batch mode.

![Simple Renamer Screenshot](SimpleRenamer.png)

## Features

- Rename files and folders in batch mode.
- Apply a variety of renaming patterns, such as adding prefixes, suffixes, replacing text, changing case, and more.
- Intuitive user interface for easy navigation and usage.

## Requirements

- .NET version 7.0 or later.
- Extended WPF Toolkit from Xceed (version 4.5.0 or later).

## Installation

1. Clone the repository: `git clone https://github.com/MawCeron/SimpleRenamer.git`
2. Open the project in your preferred .NET development environment.
3. Install the required NuGet packages, including the WPF Toolkit from Xceed.
4. Build and run the application.

## Usage

1. Launch the Simple Renamer application.
2. Select the files or folders you want to rename.
3. Choose a renaming pattern from the available options or create a custom pattern.
4. Review the preview of the new filenames.
5. Click the "Rename" button to apply the changes.
